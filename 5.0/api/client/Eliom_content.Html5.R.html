Module Eliom_content.​Html5.​R 

module R : sig..end

Creation of HTML5 content from React signals. HTML5's trees are automatically updated whenever corresponding signals change.

Content creation ¶

See Html5_sigs.​T , If you want to create an untyped form, you will have to use Eliom_content.​Html5.​D.​Raw otherwise, use the form module. For more information, see "how to make forms" 

val node : 
  'a Eliom_content.Html5.elt React.signal Eliom_pervasives.client_value ->
  'a Eliom_content.Html5.elt

Function node s create an HTML5 elt from a signal s. The resulting HTML5 elt can then be used like anyother HTML5 elt

val filter_attrib : 
  'a Eliom_content.Html5.attrib ->
  bool React.signal -> 'a Eliom_content.Html5.attrib

filter_attrib att on returns an attrib that behave like att when on is true and behave like if there was no attribute when on is false

          and type (+'a) attrib = 'a attrib
